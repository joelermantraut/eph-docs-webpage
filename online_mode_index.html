EPH: Modo Online

Búsqueda

Para realizar una búsqueda, existen dos caminos. El primero es completando el cuadro de búsqueda que indica “Buscar”, y luego presionar el botón “Buscar”, con el símbolo de una lupa a su izquierda.

Esto abre directamente el buscado con los archivos encontrados, donde en su descripción se incluya el contenido escrito en el cuadro de búsqueda. Esta búsqueda también se puede hacer presionando el botón “Enter” o “Intro”, en lugar de clickear el botón “Buscar”.

El otro camino, que permite realizar una búsqueda más detallada, es clickeando el botón “Buscar”, sin nada escrito en el cuadro de búsqueda. Esto abrirá una ventana, que nos permitirá elegir según qué parámetro buscar. Igual que en la anterior, tendremos un cuadro de búsqueda para escribir que nos interesa encontrar, y un botón “Buscar”, para comenzar la búsqueda.

Al buscar, se abre la ventana de productos, en este caso para la búsqueda “pintura”:

Funciones de ordenamiento

En esta vista se listan los productos como fueron agregados originalmente. Por defecto, están ordenados alfabéticamente según el código. Si se quiere, se puede elegir según qué columna ordenar, clickeando en los títulos de cada una.

Funciones de edición

En caso de querer modificar un producto, ya sea en código, descripción o precio, puede realizar simplemente haciendo doble click, y modificando su valor. Al presionar ENTER, este valor se modificará en la base de datos local.

NOTA: Esta modificación queda guardada en la base de datos local. La base de datos compartida NO se modifica en cada uno de estos cambios. Sin embargo, si se agrega un libro o se realiza alguna modificación que implica actualizar la base de datos compartida con la local, estas modificaciones si se compartirán.

Opciones de ayuda

Para ofrecer ayuda e información sobre EPH se ofrecen dos opciones:

  • Botón “Sobre EPH”: Que despliega información relevante respecto al software y sus condiciones de uso.
  • Botón “Ayuda”: Que al presionar, abre este mismo manual.

Control de bases de datos

Los siguientes controles corresponden al manejo de bases de datos locales.

  • Agregar planilla: Permite agregar una hoja de Excel de productos, a la base de datos local. Para poder ver un producto durante una búsqueda, es necesario primero agregarlo a la base de datos.
  • Backup de base de datos: Permite hacer una copia de esta base datos, en una carpeta elegida. Esto se puede realizar con la intención de hacer un backup, para evitar el riesgo de perder la copia actual, si está requirió varias hojas de Excel para completarse.
  • Editar base de datos: Permite revisar y eliminar todos los productos de los archivos agregados hasta el momento. Esto es útil si, por ejemplo, se agregó un libro por error, o se configuró mal, y se desea quitar de la base local.
  • Limpiar base de datos: Reinicia a 0 la base de datos actual.

A continuación, se detalla el proceso para agregar una hoja de Excel.

Agregar productos

Primeramente, presionar el botón “Agregar planilla”, presentada anteriormente. Esto abrirá una ventana de selección de archivo/s, que permitirá elegir el/los archivos que se quieran agregar a la base de datos local. Comenzaremos explicando la adición de un único archivo.

Método para una única plantilla

Luego de elegir un archivo mediante la ventana de selección de archivo/s, se presentará el siguiente menú:

En este menú, se presentan varias opciones:

  • Selección de hoja de datos: En un archivo de Excel, a veces hay más de una hoja disponible. Este menú permite elegir cual de las hojas utilizar. En el caso de la imagen, el archivo elegido presenta una única hoja, por lo cual, no se puede cambiar.
  • Columna de códigos: Letra de la columna, en Excel, donde se listan los códigos de producto.
  • Columna de descripciones: Letra de la columna, en Excel, donde se listan las descripciones de producto.
  • Columna de precios: Letra de la columna, en Excel, donde se listan los precios.
  • Inserte porcentaje de ganancia (%): El software está pensado para cargar las hojas recibidas directamente de los proveedores, donde los precios son aquellos a los que el usuario de este software los compró. Por lo tanto, este parámetro permite fijar un porcentaje adicional, que se le sumará a cada producto para que al buscar, se pueda ver el precio de venta final. Como solo se permite agregar un único porcentaje como modificador, en la sección “Cálculo de ganancia”, se detalla cómo obtener el porcentaje que se debe fijar en caso de querer agregar más de un aumento porcentual (por ejemplo, IVA + ganancia).
  • Código de proveedor: Defina un código para este proveedor, de tal manera que cuando agregue un archivo del mismo proveedor con precios nuevos (actualizados), estos se reemplazan en la base de datos local, manteniendo los productos con precios actuales, evitando así verlos por duplicado.

NOTA: En el título de la ventana, se puede leer “Configurar Excel” y entre comillas, el nombre del archivo elegido. Esto es útil en el caso de que el usuario haya olvidado la hoja elegida luego de seleccionarla.

Para continuar y agregar el libro elegido a la base de datos local, presionar el botón “Finalizar”. Esto actualizará la base de datos local.

Base de datos compartida

Luego de darle al botón “Finalizar”, en el modo online, la base de datos actualizada se subirá a la carpeta compartida en Google Drive. Esto permitirá sincronizar esta base de datos entre distintas computadoras, que tengan EPH instalado.

NOTA: Si elige “Cancelar” al momento en el que el software pide confirmación para subir la base de datos a la carpeta compartida, entonces la base de datos generada no se subirá a la carpeta en Google Drive. Como consecuencia, la base de datos compartida quedará desincronizada con la base de datos local, y estás no se volverán a sincronizar hasta que no se agregue un nuevo libro a la base de datos.

Método para múltiples plantillas

Para cargar múltiples planillas simultáneamente, el método es el mismo que para una única planilla. Solo se requieren algunas consideraciones:

  • Al abrirse la ventana de configuración de Excel, no aparecerá el menú que permite elegir desde qué hoja cargar los datos. En este modo, los datos se cargan siempre desde la primera hoja de cada planilla. Si alguna hoja está corrupta o no se puede cargar, se omite.
  • En el resto de las opciones, los valores que se elijan se aplicarán para todas las planillas. Por ejemplo, si se elige el valor A para el parámetro de códigos, se aplicará para todas las planillas indistintamente.
  • Tener presente que el código de proveedor, se aplicará para todos los productos de todos los libros cargados.

Esto es lo único a tomar en cuenta. El resto de los pasos se explican en la sección anterior.

Backup de base de datos

La opción “Backup de base de datos” permite copiar la base de datos local a otra ubicación de la computadora, con la intención de cuidar que no se sobreescriba y se pierdan los avances hasta el momento. Al presionar esta opción, se abre un ventana de selección de directorio, y al elegir “Seleccionar carpeta”, se copia el archivo y se cierra la ventana.

Editar base de datos

En esta sección, se pueden ver los libros agregados hasta el momento, y en caso de requerirlo, eliminar todos los productos de un cierto libro, de nuestra base de datos. Para acceder, presionar este botón, y esto abrirá una nueva ventana, que permitirá ver todos los libros disponibles hasta el momento:

Como venimos trabajando hasta ahora, solo hay un libro agregado. Para eliminarlo, debemos hacer doble click sobre cualquier campo de este libro. Entonces, se abrirá una ventana confirmando si se desea borrar este libro de la base de datos:

A lo que podremos responder afirmativa o negativamente. En caso de presionar “Ok”, el libro se borrará de la base de datos, y está se actualizará.

NOTA: El libro de donde se cargó en su momento, NO se eliminará.

Opciones Online

Existen dos controles adicionales específicos del modo online. Estos se pueden encontrar en la sección “Opciones Online”:

  • Carpeta compartida: Esta opción permite reasignar el enlace a la carpeta compartida donde se guarda una copia online de las bases de datos. En la sección correspondiente se detalla el proceso para obtener este enlace y cargar en la opción correspondiente.
  • Descargar bases de datos: En caso de que una copia de EPH en otra computadora haya actualizado la base de datos, puede ser requerido que la copia en la computadora actual actualice su base de datos local con la copia cargada en la carpeta compartida. Esta opción permite descargar la base de datos compartida y reemplazar la base de datos local.

Carpeta compartida

En esta sección, se detalla primero como obtener el enlace a la carpeta en Google Drive, y luego como cargar el enlace a la opción correspondiente.

Obtener enlace a la carpeta en Google Drive

Primeramente, debemos crear una carpeta. Si ya tiene una carpeta preparada para esto, puede saltarse este paso. Siga los siguientes pasos:

  1. Ingrese a Google Drive.
  2. Click en el botón “Nuevo”, ubicado arriba a la izquierda.
  3. Aparecerá un menú, entre las opciones disponibles, elegir la primera opción, “Nueva carpeta”:
  4. Aparecerá un cuadro de diálogo, introduzca el nombre de su carpeta. Elija “Crear”.
  5. Luego de crear la carpeta, esta aparecerá seleccionada, en la unidad. El siguiente paso es hacer click derecho sobre la carpeta, donde aparecerá un menú.
  6. De este menú, elegir la opción “Compartir”, entre las opciones desplegadas, la que dice “Copiar enlace”. De esta manera, el enlace quedará copiado.

Cargar enlace de la carpeta compartida

Los siguientes pasos, detallan cómo agregar el enlace obtenido a la configuración de la aplicación:

  1. Presionar el botón “Carpeta compartida”, en la sección “Opciones Online”.
  2. Esto desplegará el siguiente menú:
  3. Haciendo click derecho sobre el campo que dice “URL”, se pegará automáticamente la URL de la carpeta copiada previamente.
  4. Finalizamos presionando el botón “Actualizar”. Al hacer esto, el software analizará si el enlace provisto corresponde a un enlace de carpeta de Google Drive. De no ser así, mostrará un cartel de advertencia, y solicitará nuevamente el enlace.

Como se mencionó previamente, está opción actualizar la base de datos local con la base de datos compartida. Para esto, presione el botón “Descargar base de datos”. Se abrirá un cuadro de diálogo, pidiéndonos confirmación:

Si seleccionamos “Confirmar”, la base de datos local se reemplazará. Si por el contrario, optamos por “Cancelar”, la ventana se cerrará, cancelando el procedimiento.
