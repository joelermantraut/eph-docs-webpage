EPH: E-Commerce Product Hub

Software de gestión de productos

Objetivo

El presente documento tiene como propósito actuar como guía de uso o manual de usuario, para el software de gestión de productos EPH.

Año de publicación: 2024

Ante dudas, dificultades o errores en este manual, contactar al proveedor del producto.

Inicio

La primera ejecución del programa en una computadora nueva, puede requerir un poco más de tiempo en realizarse. Esto se debe a que el software está configurando parámetros iniciales. Así mismo, si la versión instalada incluye la función “online” (ver sección “Modalidades”, a continuación), requerirá acceder a una cuenta de Google, mediante la cual se vincularía para guardar copias de las bases de datos.

Modalidades

La aplicación se puede ejecutar en dos modos:

  • Modo "offline": El software funcionará solo con una base de datos local (almacenada en la computadora donde está instalado), y no se podrá sincronizar con otras computadoras donde también estará instalado.
  • Modo "online": Al instalarse, el proveedor debe haberlo guiado en el proceso de configuración de este modo, donde se descargaron y fijaron los archivos y carpetas necesarias para poder funcionar. Gracias a esto, EPH puede conectarse con distintas computadoras a través de internet, y compartir la base de datos generada.

A la izquierda, modo offline, a la derecha, modo online.

En este documento, se describen primero las funciones del modo offline, y luego las del modo online.

FAQs (Preguntas frecuentes)

Estás son algunas de las respuestas a preguntas frecuentes que pueden surgir durante el uso del software.

El software no se abre

Si el software fue correctamente instalado, y luego de algún cambio o movimiento en el dispositivo dejó de funcionar, es posible que la causa del problema sea una corrupción del mismo programa, o del directorio de instalación. En estos casos, se recomienda correr el instalador adjunto con la instalación original, y si continúan las dificultades, contactar al proveedor.

La corrupción del serial puede deberse a un cambio en los componentes de la computadora utilizada. En este caso, el proveedor se compromete a re-instalar el software, cobrando el servicio nuevamente, más no el producto.

¿Qué pasa si por error desinstalé el software?

Al desinstalar el software, el programa y todos sus archivos son borrados del sistema. La única excepción, es el archivo serie, que permanece guardado en la carpeta de instalación. Luego de la primera instalación, el encargado deberá dejarle un instalador, con el cual, podrá volver a instalar el software. Este instalador no incorpora una clave serial, por lo cual, si intenta instalarlo en una computadora no licenciada, este no funcionará. Por el contrario, si su computadora fue correctamente licenciada, el software se instalará correctamente.
